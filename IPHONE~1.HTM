แนวทาง Improve Performance
  • BMA แข็งแรง (>90%) — รักษาโมเมนตัมและถอดบทเรียนไปปรับใช้กับ UPC

Shop Type

Performance · Matching / Non-Matching / Hall / COCO

แนวทาง Improve Performance
  • Matching เป็นเครื่องยนต์หลัก (>100%) — ต่อยอดด้วยแคมเปญ upsell รุ่นระดับบน
  • DTAC HALL/COCO ต่ำกว่าเป้ามาก ควรเทรนพนักงานขายและทบทวน assortment ของรุ่นที่วางขาย
  • เติม/ย้ายสต็อกรุ่นขายดีเข้าช่องทาง dtac เดิม เพื่อปลดล็อกโอกาสยอดขายที่ยังค้างอยู่
